О проекте

Создание модели генератора для автономного энергоснабжения в удаленных условиях на основе термоэлектрических эффектов.

Теоретическая часть

Элемент Пельтье

Термоэлектрический модуль, преобразующий разницу температур в электричество.

Эффект Пельтье

Выделение/поглощение тепла при прохождении тока через контакт разных материалов.

Эффект Зеебека

Генерация напряжения при разнице температур в цепи из разных проводников.

Практическая часть

Материалы необходимые для реализации:

  • Термоэлектрический модуль Пельтье
  • Преобразователь напряжения
  • Свечки
  • радиатор
  • термопаста
  • Алюминиевые профили(необязательно)

Этапы сборки:

  • Соединяем элементы Пельтье последовательно для эффективной работы при малом нагреве.
  • Для стабильной работы генератора необходимо подключить повышающий преобразователь напряжения DC-DC с выходным напряжением 5в
  • Элементы Пельтье соединяем с радиатором и алюминиевыми профилями с помощью термопасты и винтов со стяжками, а также прикрепляем преобразователь к корпусу, чтоб он не болтался
  • Прикрепляем к концам радиатора пластмассовые элементы для подставки, чтобы провода не соприкасались с подставкой
  • Изготавливаем отдельно подставку из алюминиевых уголков размером: 125х170м

Результаты

Сравнительная характеристика

Разработанная модель генератора демонстрирует конкурентоспособные характеристики в сравнении с коммерческими аналогами. Несмотря на меньшую мощность и время зарядки, она обладает такими преимуществами, как низкая стоимость, легкость и простота сборки. Это делает ее перспективной для использования в местах, где доступ к традиционным источникам энергии ограничен.

Заключени

В результате создания нашей модели термоэлектрического преобразователя, на основе полученных данных, я считаю, что элемент Пельтье доказал свою эффективность в качестве альтернативного источника энергии , пока только для маломощных устройств. Разработанная модель может быть использована в бытовых условиях, Например, для использования фонариков или радиоприемников в походных условиях или в местах с отсутствием доступа к электричеству. Несмотря на относительно низкий КПД, простота и экологичность технологии делают ее перспективной для дальнейшего развития и применения.
